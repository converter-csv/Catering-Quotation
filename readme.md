# Thong Kee Catering Quote Generator

An interactive, single-page web application designed for **Thong Kee Malaysia Kopitiam** to generate catering quotes. This tool allows users to select catering packages, customize menu items, and generate a professional summary for printing or saving as a PDF.

## 🚀 Features
- **Dynamic Package Selection:** Choose between different catering tiers (e.g., Package A, B, or C) with real-time price updates.
- **Interactive Menu:** Select from a variety of categories including Noodles, Rice, Fried Bites, Vegetables, and Signature Drinks.
- **Smart Validation:** Automatic counters ensure you select the correct number of items allowed by your chosen package.
- **Customer Information Form:** Fields for event details, venue, date, and contact info.
- **Professional Quote Summary:** Generates a clean, print-friendly summary including grand totals and deposit requirements.
- **Print to PDF:** Built-in print optimization for physical records or digital sharing.

## 🛠️ Built With
- **HTML5/CSS3:** Responsive design with a "Kopitiam" aesthetic using custom fonts (Playfair Display, DM Sans).
- **JavaScript (Vanilla):** Logic for quantity calculations, package switching, and quote generation.

## 💻 How to Use
1. Clone this repository or download the `index.html` file.
2. Open `index.html` in any modern web browser.
3. Fill in the customer details and select a catering package.
4. Pick your desired dishes using the quantity selectors.
5. Click **"Generate Quote Summary"** to review.
6. Use the **"Print / Save as PDF"** button to export your quote.
